# Roadmap

RootLabs POS for WooCommerce is in an early public release stage.

The initial goal is to provide a useful, auditable, open source POS foundation for WooCommerce stores.

## Current focus

- Public repository cleanup.
- Documentation for installation and operation.
- Stable release packaging.
- Safer onboarding for new stores.
- Clearer compatibility notes for WordPress, WooCommerce, PHP, and HPOS.

## Planned improvements

- Better product search performance for large catalogs.
- Cleaner refund and order search workflows.
- More configurable receipt and ticket layout.
- Optional RootLabs credit visibility in printed tickets.
- Improved settings UX.
- Translation files.
- More automated tests.
- Public issue templates.
- Developer documentation for hooks and extension points.

## Not planned for the first public release

- Cloud sync.
- SaaS dashboard.
- Multi-store cloud reporting.
- Hardware-specific integrations beyond browser-based printing.

## Philosophy

The plugin should remain practical, auditable, and useful for small and medium WooCommerce stores. Paid plugins exist, but a serious open source alternative should also exist.
